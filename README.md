# BOINC (Forked)

- **[History](doc/history/)**: Archived `checkin_notes` and implementation logs.
- **[Planning](doc/planning/)**: Legacy TODOs and scaffolding.

## Build & Install

### Windows

Open `win_build/boinc.sln` in Visual Studio.

### Unix/Linux

Standard Autotools build:

```bash
./_autosetup
./configure
make
```
